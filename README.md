## Run Locally

Clone the project

```bash
  git clone https://github.com/deshmukh612/Naukri-Clone.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
